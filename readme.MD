# Sistem Partai PRO

Demo sistem digital untuk:
- Aktivasi kader
- Layanan masyarakat
- Monitoring aktivitas

## Fitur
- Form laporan masyarakat
- Dashboard kader
- Sistem poin
- Leaderboard

## Tujuan
Membantu partai bergerak aktif setiap hari dan berbasis data.

## Cara Jalankan
Buka file index.html di browser.
